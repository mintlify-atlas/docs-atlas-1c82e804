## Overview

Line charts connect data points with straight or curved lines, making them ideal for displaying trends over time or continuous data. They're one of the most versatile chart types and can be combined with point markers, area fills, and various styling options. TeeChart offers both regular line series and fast line series optimized for large datasets.

**Use cases:**
- Showing trends over time
- Displaying continuous data
- Comparing multiple data series
- Real-time data monitoring
- Scientific and technical data visualization

## Series Classes

**TLineSeries** - Standard line charts with full features
**TFastLineSeries** - Optimized for large datasets and real-time data
**THorizLineSeries** - Horizontal line charts

### Key Properties

| Property | Type | Description |
|----------|------|-------------|
| `Pointer` | TSeriesPointer | Configure point markers |
| `LinePen` | TChartPen | Customize line appearance |
| `Stairs` | Boolean | Display as stepped lines |
| `Smoothed` | Boolean | Apply curve smoothing (Bezier) |
| `Stacked` | TCustomStack | Stack multiple series |
| `OutLine` | TChartPen | Add outline effect |
| `TreatNulls` | TTreatNullsStyle | How to handle null values |
| `DrawAllPoints` | Boolean | Draw all points (FastLine only) |
| `DrawAllPointsStyle` | TDrawAllPointsStyle | Point drawing style (FastLine) |

### Fast Line for Real-Time Data

From VCL/TeeNew/FastLine_Realtime.pas (example pattern):

```pascal
uses
  TeEngine, Series;

var
  FastLine: TFastLineSeries;
  
procedure SetupRealtimeLine;
begin
  FastLine := TFastLineSeries.Create(Chart1);
  Chart1.AddSeries(FastLine);
  
  // Optimize for real-time performance
  FastLine.DrawAllPoints := True;
  Chart1.AutoRepaint := False;  // Manual repaint for better performance
end;

procedure AddRealtimePoint(YValue: Double);
begin
  FastLine.Add(YValue);
  
  // Limit points to last 1000
  if FastLine.Count > 1000 then
    FastLine.Delete(0);
  
  Chart1.Invalidate;  // Manual repaint
end;
```

## Performance Optimization

### Use FastLine for Large Datasets

```pascal
var
  FastLine: TFastLineSeries;
begin
  FastLine := TFastLineSeries.Create(Chart1);
  
  // Performance settings
  FastLine.DrawAllPoints := False;  // Skip points when zoomed out
  Chart1.AutoRepaint := False;
  Chart1.Axes.FastCalc := True;
end;
```

### Optimize Real-Time Updates

```pascal
procedure OptimizeForRealtime;
begin
  Chart1.AutoRepaint := False;
  Chart1.BufferedDisplay := True;
  
  // Disable automatic axis scaling
  Chart1.LeftAxis.Automatic := False;
  Chart1.LeftAxis.SetMinMax(0, 100);
end;
```
